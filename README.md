# Modern Web Developer Portfolio

A premium, high-performance portfolio website built with React 19 and Tailwind CSS 4. This project showcases my professional journey, technical expertise, and a curated selection of full-stack and CMS-based projects.

🔗 **Live Demo:** [https://mamun-ten.vercel.app/](https://mamun-ten.vercel.app/)

---

## ✨ Features

- **Modern UI/UX:** Sleek dark-themed design with coral accents, vibrant gradients, and professional typography.
- **Dynamic Animations:** Interactive floating tech icons, glassmorphism effects, and smooth scroll transitions.
- **Project Showcase:** Categorized display of MERN stack, Next.js, and WordPress projects with live links and repository access.
- **Fully Responsive:** Optimized for all screen sizes, from mobile devices to ultra-wide monitors.
- **SEO Optimized:** Semantic HTML structure and optimized performance for better search engine visibility.

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/mamun-jsx/Portfolio.git
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd Portfolio
    ```

3.  **Install dependencies:**

    ```bash
    npm install
    ```

4.  **Start the development server:**
    ```bash
    npm run dev
    ```
